- Chủ đề: Thực hành ngoại vi I2C trên STM32
- Chức năng Gửi dữ liệu từ I2C1 sang I2C2 trên STM32
- Cách sử dụng:
1. Mở file cấu hình (.ioc) để xem cấu hình STM32,  Nối các chân I2C đúng cấu hình.
2. Gắn dây nạp STM32.
3. Mở file project trên STM32CubeIDE để compile và chạy chương trình.
4. Trong chế độ Debug, gán biến trạng thái và biến buffer để giám sát quá trình gửi dữ liệu .
